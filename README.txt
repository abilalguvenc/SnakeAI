#########################################################################################
##                                      TÜRKÇE                                         ##
#########################################################################################

"Visual Studio Code" ve "Python 3.7.6" yazılımlarını yükle.
NOT: Python 3.8 sürümleri Tensorflow kütüphanesini desteklememektedir.

Gerekli olan bu üç kütüphaneyi yükleyin: numpy, pygame, tensorflow
 > pip install numpy
 > pip install pygame
 > pip install tensorflow

Proje dosyasını "Visual Studio Code" ile açın ve aşağıdaki satırı terminale yazın.
 > python main.py
 $ Select your choice
 $ 1. Load Agent
 $ 2. Train Agent
 $ Choice:
Kayıtlı ajanları yüklemek için "1"i seçin, "501", "5001" veya kayıtlı olan bir ajan numarası yazarak ajan seçiminizi yapın.
 $ Select Agent ID: agent_
 > 501


#########################################################################################
##                                      ENGLISH                                        ##
#########################################################################################

Install "Visual Studio Code" and "Python 3.7.6".
NOTE: Python 3.8 versions doesn't support Tensorflow lib.

Install following libraries: numpy, pygame, tensorflow
 > pip install numpy
 > pip install pygame
 > pip install tensorflow

Open this folder with "Visual Studio Code" and type following line to the terminal.
 > python main.py
 $ Select your choice
 $ 1. Load Agent
 $ 2. Train Agent
 $ Choice:
To load agents select choice "1" and following type "501" or "5001" or any other number that is available to following line.
 $ Select Agent ID: agent_
 > 501
